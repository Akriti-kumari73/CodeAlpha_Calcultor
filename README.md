# CodeAlpha_Calcultor
This project  is a Basic calculator web application built using html,css and javascript with all code seprated into individual files .
It perform standard arithmetic operations  such as addition ,substraction, multiplication, and division , and includes features like clearing the display and handling user input through interactive buttons.The calculator has a clean user interface designed with CSS, and javaScript handles all calculation logic.
This project is ideal for beginners learning DOM manipulation ,CSS layout , and event handling in javaScript.
